# Client Lead Management System (Mini CRM)

This project is a simple Mini CRM (Client Relationship Management System) developed as part of my Full Stack Web Development Internship.

## 📌 Project Overview
The application helps in managing client leads generated from different sources. It allows adding, updating, and deleting leads with their status and notes.

## 🚀 Features
- Add new leads (Name, Email, Source, Status, Notes)
- View all leads in a structured format
- Update lead status (New / Contacted / Converted)
- Edit notes for each lead
- Delete leads
- Data stored using localStorage (no backend required)
- Responsive design (mobile and desktop)

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 💡 Concept Used
- CRUD Operations (Create, Read, Update, Delete)
- Local Storage for data persistence
- Basic UI/UX design principles

## ⚠️ Note
This project simulates backend functionality using localStorage instead of an actual database.
